# Picatti Apparel App

This folder is ready to publish on GitHub Pages.

## Publish steps
1. Create a new public repository on GitHub named `picatti-apparel`
2. Upload the files from this folder to the repository root
3. In GitHub, go to **Settings** > **Pages**
4. Under **Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
5. Save
6. Your live link will be:

`https://dougpicatti-ctrl.github.io/picatti-apparel/`

GitHub Pages can take a minute or two to go live.

## What is included
- `index.html` — the full app

## Notes
- Orders and admin settings are stored in the browser's local storage on each device.
- For real multi-user shared data later, this would need a backend/database.
